 📖 Sarah J. Maas Web Tribute (Certification Project)

This repository contains the source code for my 12th Grade Computer Science Certification. The project is a responsive multi-page website dedicated to the works of fantasy author Sarah J. Maas.

 📂 Project Overview

The goal of this project was to demonstrate proficiency in frontend web development by creating an immersive, themed user interface. The site serves as a digital portfolio for the author's three major book series: "Throne of Glass", "A Court of Thorns and Roses", and "Crescent City".

 🚀 Features & Pages

The website is structured into five distinct sections:

* Home: Landing page featuring a navigational overview.
* Books: A catalog layout detailing series books.
* Author: A biographical section highlighting the author's career and achievements.
* Quotes: A visually styled gallery showcasing key literary passages.
* Contact: An interactive form designed for user engagement.

 🛠️ Technologies Used

* HTML5: Semantic structure and layout.
* CSS3: Custom styling, Flexbox/Grid layouts, and responsive design media queries.

 📦 How to Run

1.  Clone this repository:
    ```bash
    git clone [https://github.com/amabarbus/web-page.git](https://github.com/amabarbus/web-page.git)
    ```
2.  Navigate to the project folder.
3.  Open `home.html` in your browser to view the site.
